# yolgezer

Your monero node on the road.
Shell scripts to automate installing a monero node
on your android termux.

## Install

```
bash -c "$(curl -fsSL https://github.com/k4r4b3y/yolgezer/raw/main/src/yolgezer.sh)"
```
